From Prompt to Paper — AI-Automated JAMA Workflow
Screen cues mapped to the voice-over script
--------------------------------------------------
Format:  WHEN YOU HEAR — the spoken trigger phrase
         SCROLL TO     — which tutorial section / element
         ACTION        — mouse or zoom move to do

No timestamps. Scroll when you hear the trigger phrase, regardless of speed.

RECORDING TIPS
- 1920×1080 browser window, zoom at 90% so sections don't need mid-sentence scrolling.
- Record browser + Finder panels using QuickTime (File → New Screen Recording) or OBS.
- Lower-third text labels recommended when switching sections, e.g. "Stage 3 — Analysis."
- Add chapter markers in your editor (iMovie / DaVinci) matching the cue numbers above.
